# Fluent::Plugin::Ufw

A fluent plugin to parse ufw log

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'fluent-plugin-ufw'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fluent-plugin-ufw
